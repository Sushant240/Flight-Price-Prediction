# Flight Price Prediction

## Project Overview
Built a machine learning regression model to predict airline ticket prices using travel and booking information.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Features
- Data cleaning
- Datetime feature engineering
- EDA
- Regression modeling
- Hyperparameter tuning

## Algorithms Used
- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

## Outcome
Developed an accurate flight fare prediction system using machine learning techniques.
